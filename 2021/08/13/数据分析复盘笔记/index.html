记录 数据分析 复盘笔记

day1 2021年8月13日 19点39分

value_counts()

  • Series类型函数,df必须指定对应的列
  • sort
    • 默认降序排列
  • ascending
    • 默认 False
  • normalize
    • 默认 false
    • 如若为True,则转换为百分比

map()

  • series类型的函数
  • 作用于series中的每一个数据

apply()

  • 作用于DF中的每一行或者每一列

applymap()

  • 作用于DF中的每一个元素

nunique()

  • series类型的函数
  • 查找独一无二的元素(自动去重)

查找某一列中以’G’开头的元素

  • df[df[‘col’].str.startswith(‘G’)]
    • .str
      • 可以将元素转化为字符串，进而使用字符串中的函数

loc与iloc的区别

  • loc
    • locatiion的意思
    • 按标签取数据，行标签或者是列标签
    • 上述传输的参数都是标签
  • iloc
    • integer,所以只能接受整数作为参数
    • 不同于loc,iloc按照位置去选取数据。所以传入的参数是位置，先行后列

isin()

  • df.isin([‘a’, ‘b’])
    • 判读df中的所有数据，如果为a或者b则返回True,否则返回False
    • df可以根据布尔值进行取数
    • 类似于sql中的 in,代表or的意思
  • df.col.isin([‘a’, ‘b’])
    • 判断df中col列中的所有数据，如果为a或者b则返回True,否则返回False

drinks.groupby(‘continent’)

  • 是一个groupby对象
  • 后面可以选择对应的列，也可以对整体数据进行运算
  • 之后可以跟聚合函数，如果是多个聚合函数，可以使用.agg(),传入对应的聚合函数即可

sql

  • 针对表内容做修改，使用update … set
  • 针对表的结构做修改，使用 alter…
  • replace(‘字符串’, 待替换字符，替换为字符)
  • right(‘字符串’, num)
    • 返回字符串从右边开始num个字符
  • group_concat()

excel

函数

  • median()
    • 中位数
  • mode()
    • 众数

日期函数

  • today()
    • 返回当天的年月日
  • now()
    • 返回当前时间

文本函数

  • &

    • 联结不同单元格
    • 也可以指定某个连接符

  • text()

    • 字符串格式化
    • 按照某种输出格式进行显示

  • VLOOKUP()

    • 查找内容
    • 查找区域
    • 所在区域的列数
    • 精确匹配or模糊匹配

  • IFERROR()

    • 用来处理错误，如果有错误显示什么，没有错误显示什么

  • SUMIFS()

    • 待求和区域

    • 待判断区域，条件2

  • COUNTIFS()

    • 待判断区域，条件1
    • 待判断区域，条件2

  • AVERAGEIFS()

    • 待求平均区域
    • 待判断区域，条件1
    • 待判断区域，条件2

假设连续型随机变数取值介于a-b之间的概率为

P(a<=X<=b)

  • 变量X的概率密度函数
    • 通过概率密度函数进行特定区域的积分，可以求出在该区域下的概率

连续型随机变数的平均值和方差

正态分布

呈现带有误差现象的数据大多可用正态分布来表示

  • 均值为0，标准差为1的正态分布称为标准正态分布

标准正态分布性质

用统计学研究事物时，一般来说有95%概率发生的事件被视为发生概率较大的事件，除此之外的事件被视为异常
