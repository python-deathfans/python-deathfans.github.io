本篇内容整理 最近学习的 浏览器高级检索技巧以及浏览器的一些使用。

  • 标签页切换
    • ctrl+tab
      • 顺序切换(先进后出)
    • ctrl+shift+tab
      • 逆序切换(先进先出)
  • 检索词 辅助词
    • 检索词 thunder://
      • 检索对应的大文件
      • 检索词 == 电影
        • 返回电影下载连接
      • 检索词 == 软件名字
        • 返回软件下载地址
    • 检索词 1080
      • 返回电影下载链接
  • 模糊搜索
    • *
    • 罗曼蒂克* 葛优
      • *可以代替任意多个字符
  • 精确搜索
    • “关键词”
    • 搜索引擎不会拆分关键词，也不会更换位置，总之原模原样出现在网页标题或者正文中
  • 电商网站
    • 海量学习资源
    • 口语化
    • 易上手
    • 免费
