pycharm

ctrl + alt + L : 代码格式化

调试模式

  • step over(F8)
    • 跳转到下一个断点(如果函数内部存在)，否则跳过整个函数
  • step into(F7)
    • 跳进函数内部
    • 单步执行
  • step out()
    • 将函数剩余部分全部执行完毕
  • run to cursor
    • 把光标当作一个断点
    • 常用于一步执行循环体，只见结果，不见过程

python

闭包&装饰器

闭包条件

  • 外函数内部定义了内函数
  • 内函数使用了外函数中的变量
  • 外函数的返回值是内函数

装饰器

  • 在不改变函数主体功能的情况下增加新的功能

  • 使用闭包

  • 计算函数运行时间

    • import time


def wrapper(func):
    def inner(*args, **kwargs):
        print("i am decorated!!")

        start_time = time.time()
        res = func(*args, **kwargs)
        end_time = time.time()
        used_time = end_time - start_time

        print(f'执行work使用了{used_time}s')

        return used_time
    return inner


@wrapper
def work():
    for i in range(10):
        print(f'i am working{i}!!')
        time.sleep(1)
