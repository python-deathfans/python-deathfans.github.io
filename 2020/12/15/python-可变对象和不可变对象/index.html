python-可变对象和不可变对象

发表于2020-12-15|更新于2020-12-18|python
|字数总计:338|阅读时长:1分钟|阅读量:

本篇文章记录一下python中的可变对象和不可变对象，以及为什么没法使用可变对象当作函数的默认参数。

区别

  • 可变不可变指的是内容

分类

  • 可变对象
    • list
    • dict
  • 不可变对象
    • int
    • str
    • float
    • tuple

不可变对象

  • python中的变量存放的是对象引用，对于不可变对象而言，尽管对象本身不可变，但是对象的引用可以改变。

可变对象

  • 对象内容可以发生变化。当对象的内容发生变化时，引用不会发生变化

为什么没法使用可变对象当作函数的默认参数

python中万物皆对象，函数也是一个对象，也有自己的属性。

def 是一条可执行语句，python解释器执行def语句时，就会在内存中创建一个函数对象(此时，函数里面的代码逻辑并不会执行，因为还没有调用)，在全局命名空间，有一个函数名(变量叫func)会指向该函数对象，至此开始，不管函数调用多少次，函数对象只有一个，不会因为调用而出现多个函数对象。函数对象生成之后，它的属性：名字和默认参数列表都将初始化完成。
